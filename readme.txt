== Description ==
Herosense is the perfect adition for the tech-savy devleoper or plain old front blogger. It has a blog style layout similar to medium where the reader is engaged in reading the content without any disturbances. Herosense features a high hero area where you can set up your own featured image, stlyed independently from the post page. You can choose your own background images on the index page and custom colors.

For more information about Herosense please go to http://creativecoon.com/hero.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Twenty Fourteen in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://creativecoon.com/hero for a guide to customize this theme.
5. Navigate to Appearance > Customize in your admin panel.
